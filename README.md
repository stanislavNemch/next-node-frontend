# ToolRent — сервіс оренди інструментів

## Опис
Сервіс дозволяє користувачам публікувати інструменти, переглядати каталог, 
фільтрувати, бронювати та залишати відгуки.

## Технології
- Next.js 15 (App Router)
- TypeScript
- CSS Modules
- Zustand
- React Query (TanStack)
- Formik + Yup
- Axios
- next/image
- паралельні маршрути (modal routing)

## Запуск
npm i  
npm run dev
